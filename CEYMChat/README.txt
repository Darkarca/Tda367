
To run the server:
	Run the RunServer.sh or RunServer.bat file
To run the client:
	Run the RunClient.sh or RunServer.bat file
To build the project with Maven:
	Run the Assembly plugin with lifecycle:assembly



********************************************************************************
-Multiple clients can be run on the same machine.
-The Server must run before a client logs in.
-To run clients on a machine different to the one running the server, contact us.
